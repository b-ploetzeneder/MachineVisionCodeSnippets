## Inputs


| Name                                        | Description                                           |
| ------------------------------------------- | ----------------------------------------------------- |
| Kernel Size|  defines how big the kernel should be. I automatically turn it into an odd nr | 

## Outputs

| Name                                        | Description                                           |
| ------------------------------------------- | ----------------------------------------------------- |
| Kernel | Highlight Kernel of any size |
